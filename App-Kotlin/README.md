## Ambiente de desenvolvimento utilizado

Estes códigos foram escritos no *Android Studio*, *IDE* oficial para desenvolvimento de aplicativos *Android*. 


## Projetos presentes nesta pasta

1. VoiceSurgeryGoogleAPI

   Versão inicial utilizando API do *Google SpeechRecognizer*, nativa do Android para transcrição de áudios.

2. VoiceSurgeryWhisper

   Projeto adaptado para transcrição com o Whisper, enviando o áudio para o servidor que usa a opção *open source*. As demais funcionalidades são semelhantes às do projeto acima.
